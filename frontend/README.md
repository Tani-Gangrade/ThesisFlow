This is a Next.js frontend for ThesisFlow MARA.

## Getting Started

Set backend URL (optional, defaults to `http://127.0.0.1:8000`):

```bash
export NEXT_PUBLIC_API_BASE_URL=http://127.0.0.1:8000
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000/chat](http://localhost:3000/chat).

Upload a PDF, then chat with memory and RAG enabled.
